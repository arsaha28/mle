Setup Guide

Run the Tutorial Locally

Everything you need to get from zero to running your first file.

0

What you need before starting

Make sure these are installed on your machine.

🐍Python3.11 or higher
📦pipcomes with Python
🖥️VS Codeany recent version
🔑OpenAI API keyplatform.openai.com
🌿Gitto clone the repo
Check your Python version — open a terminal and run python3 --version. You need 3.11+. If you have an older version, download the latest from python.org.
1

Clone the repository

Download the code to your local machine.

Terminal
git clone https://github.com/arsaha28/mle.git
cd mle
2

Create a virtual environment

A virtual environment keeps this project's packages isolated from the rest of your system. Always do this — it prevents version conflicts between projects.

macOS / Linux
python3 -m venv venv
source venv/bin/activate
Windows (Command Prompt)
python -m venv venv
venv\Scripts\activate
Windows (PowerShell)
python -m venv venv
venv\Scripts\Activate.ps1
You'll know it worked when your terminal prompt shows (venv) at the start.
3

Install dependencies

One command installs everything the tutorial needs.

Terminal (inside venv)
pip install -r requirements.txt

This installs the following packages:

  • langchain >= 0.3.0
  • langchain-openai >= 0.2.0
  • langchain-community >= 0.3.0
  • langchain-text-splitters >= 0.3.0
  • langgraph >= 0.2.0
  • faiss-cpu >= 1.8.0
  • chromadb >= 0.5.0
  • pypdf >= 4.0.0
  • pydantic >= 2.0.0
  • python-dotenv >= 1.0.0
  • tiktoken >= 0.7.0
  • unstructured >= 0.14.0
Slow install? chromadb and unstructured pull in large dependencies. This is normal — it can take 2–3 minutes on first install.
4

Set up your OpenAI API key

The tutorial calls the OpenAI API (gpt-4o-mini and text-embedding-3-small). You need a key to authenticate.

Step 4a — Copy the example env file:

macOS / Linux
cp .env.example .env
Windows
copy .env.example .env

Step 4b — Open .env in VS Code and replace the placeholder:

.env
OPENAI_API_KEY=sk-...your-actual-key-here...
Never commit your .env file. It contains your secret key. The .gitignore already excludes it — don't add it manually.
Get a key at platform.openai.com → API Keys → Create new secret key. You'll need to add a small credit balance (~$5) to make API calls.
5

Open in VS Code

Open the project folder and point VS Code at the virtual environment you created.

Terminal
code .

Then select the Python interpreter:

  • Press Cmd+Shift+P (Mac) or Ctrl+Shift+P (Windows)
  • Type: Python: Select Interpreter
  • Choose the one that shows ./venv/bin/python or .\venv\Scripts\python.exe
Recommended VS Code extensions: Python (Microsoft) — syntax, linting, debugger  |  Pylance — type checking and autocomplete. Install both from the Extensions panel (Ctrl+Shift+X).
6

Run your first file

Start with file 01 and work through them in order — each builds on the last.

Terminal (from the project root)
python tutorial/01_basic_llm.py
Expected output: you'll see the model's response printed to the terminal. If you see an AuthenticationError, double-check your API key in .env.
